# Blog Application

## URL
https://railsguides.jp/getting_started.html

## 前提条件
* 以下条件にて開発
    * Ruby version: 2.3.4
* 以下をインストール済み
    * git
    * bundler

## インストール
```
$ git clone git@github.com:torimoto-shinya/blog.git
$ cd blog/
$ bundle install --path vendor/bundle
```

## 動作確認
```
$ bin/rails server
```
